CS 832 - HCI - Gaze Analytics Dashboard

Prototype : preview.html
Homework 3 Report : report.pdf

Instructions:

  1. To open the application in web browser, click on preview.html
  2. Login in to the application with your username and password.
  3. If you don't have an account click on "Sign Up Free!" link to create an account.
  4. Once you signed up and logged in, you can select your view.
    • Select Participant View - To view the video of and record their eye movements for further analysis.
    • Select Presenter View - To analyze the visual attention measures generated by the application based on participants' eye movements

Participant View
  1. Start the calibration, the screen will start a sequence of targets for calibration. Stare at the target so that the application can sample from the camera input for calibration.
  2. Once calibration is completed, Start the view to record eye tracking.
Presenter View
  1. To analyze the recorded eye movement, go to Analytics Dashboard.
  2. Select the recording to analyze.
  3. Select the visualization type from the drop-down.
